## i 茅台 APP 小茅运活动

> nodejs 实现小茅运旅行活动自动执行

### TODO

### 安装

```shell
npm i -g pm2
git clone git@github.com:mrabit/xmtravel.git
cd xmtravel
npm install
```

### 启动

```shell
cp config.example.js config.js
pm2 start script.json
```
